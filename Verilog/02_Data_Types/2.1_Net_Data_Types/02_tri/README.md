# TRI Net Data Type

## Objective

Learn the `tri` net data type in Verilog, understand its behavior, and compare it with the `wire` net type.

---

## Topics Covered

- What is `tri`?
- Why `tri` is used
- Syntax
- Scalar `tri`
- Vector `tri`
- Multiple drivers
- Comparison with `wire`
- Practical applications

---

## Files

| File | Description |
|------|-------------|
| `01_tri_declaration.v` | Basic `tri` declaration |
| `02_scalar_tri.v` | Scalar `tri` example |
| `03_vector_tri.v` | Vector `tri` example |
| `04_multiple_driver.v` | Demonstrates multiple drivers |
| `notes.md` | Detailed learning notes |

---

## Learning Outcomes

After completing this module, you will be able to:

- Declare `tri` nets
- Understand how `tri` differs from `wire`
- Use `tri` in shared bus connections
- Recognize situations involving multiple drivers
