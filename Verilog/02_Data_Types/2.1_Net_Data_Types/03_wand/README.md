# WAND Net Data Type

## Objective

Learn the `wand` (wired-AND) net data type, understand its resolution behavior, and compare it with `wire` and `tri`.

---

## Topics Covered

- What is `wand`?
- Syntax
- Resolution rule
- Scalar `wand`
- Vector `wand`
- Multiple drivers
- Comparison with `wire` and `tri`

---

| `notes.md` | Detailed notes |

---

## Learning Outcomes

After completing this module, you will be able to:

- Declare `wand` nets.
- Understand wired-AND resolution.
- Predict the result of multiple drivers.
- Differentiate `wand` from `wire` and `tri`.
